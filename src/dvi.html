Smart_Calculator

About programm

The program is implemented on the gkt4 graphics library. The calculator has the ability to calculate the result of the entered expression. if there is an x, the program will offer to replace it with a numeric value and calculate the result taking into account this value.

The user has the ability to plot the entered expression. before plotting, you will be prompted to set the scale along the X and Y axes.

How to start

To start using the calculator, you need to install it using the make install command. this command will install the program on the desktop in the SmartCalc folder.

Unistall

to remove the user program. can use the make uninstall command
